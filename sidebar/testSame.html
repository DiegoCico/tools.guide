testSame Function

The testSame() function is designed to compare the output of a given script with an expected value and report the result. It is useful for writing unit tests where you need to assert the output of certain operations.

Usage Example

// Sample methods to be tested
public static int addNumbers(int a, int b) {
  return a + b;
}

public static int multiplyNumbers(int a, int b) {
  return a * b;
}

public static String concatenateStrings(String a, String b) {
  return a + b;
}

Expected Output

Test (AdditionTest): Passed
Test (MultiplicationTest): Passed
Test (ConcatenationTest): Passed

Function Definition

public static <T> void testSame(Supplier<T> script, T expectedOutput, String name) {
    T actualOutput = script.get();
    if (actualOutput == null ? expectedOutput == null : actualOutput.equals(expectedOutput))
        System.out.println("Test (" + name + "): Passed");
    else
        System.out.println(String.format("Test (%s): Got = %s, Expected = %s", name, actualOutput, expectedOutput));
}

Parameters

  • script: A Supplier<T> that returns the output to be tested.
  • expectedOutput: The expected output value to compare against.
  • name: A descriptive name for the test case.

Expected Output

The function will print "Test (name): Passed" if the actual output matches the expected output, otherwise it will print what was received versus what was expected.

Implementation Notes

  • This function uses Java generics and is suitable for any type of data.
  • It's especially useful in scenarios where testing the outcome of functional interfaces or lambdas.
