isAInteger Function

The isAInteger() function checks if a given string can be successfully converted to an integer without causing an exception.

Usage Examples

>>> String validNumber = "12345";

>>> String invalidNumber = "123.45";

>>> System.out.println(isAInteger(validNumber));

true

>>> System.out.println(isAInteger(invalidNumber));

false

Function Definition

public static boolean isAInteger(String str) {
    try {
        Integer.parseInt(str);
        return true;
    } catch (NumberFormatException e) {
        return false;
    }
}

Parameters

  • str: The string to be checked.

Return Type

Returns true if the string can be converted to an integer, and false otherwise.

Notes

  • This method is useful for validating user input or parsing data where integer values are expected.
  • It utilizes a try-catch block to handle potential NumberFormatException.
