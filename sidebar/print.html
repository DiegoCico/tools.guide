print Function

The print() function in Java is used to display text or variables in the terminal. Unlike println(), it does not append a newline at the end of the output.

Syntax

System.out.print(Object obj);

Parameters

  • obj: The object to be printed. It can be a string, number, or any other data type.

Return Type

This method does not return any value.

Usage Example

>>> System.out.print("Hello World!");

Hello World!

>>> String name = "Gaby";

>>> int age = 19;

>>> System.out.print(name + " is " + age + " years old");

Gaby is 19 years old

Notes

  • For printing with a newline, use System.out.print().
  • This method can be overloaded with different types of parameters.
  • For formatted output, consider using System.out.printf().
