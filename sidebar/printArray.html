printArray Function

The printArray() function is designed to print the elements of an array in a readable format. It iterates over the array and prints each element.

Usage Examples

>>> int[] numbers = {1, 2, 3, 4, 5};

>>> printArray(numbers);

1, 2, 3, 4, 5

The function simplifies the process of printing array elements, without needing to write a loop each time.

>>> String[] names = {"Alice", "Bob", "Charlie"};

>>> printArray(names);

Alice, Bob, Charlie

Function Definition

public static <T> void printArray(T[] array) {
    for (T element : array) {
        System.out.print(element + ", ");
    }
    System.out.println();
}

Notes

  • This method can handle arrays of any type, including primitives and objects.
  • It uses a generic type T, making it versatile for various array types.
