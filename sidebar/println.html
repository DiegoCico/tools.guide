println Function

The println() function in Java is used to display text or variables in the terminal and appends a newline at the end of the output.

Syntax

System.out.println(Object obj);

Parameters

  • obj: The object to be printed. It can be a string, number, or any other data type, followed by a newline.

Return Type

This method does not return any value.

Usage Example

>>> System.out.println("Hello World!");

Hello World!

>>> String name = "Gaby";

>>> int age = 19;

>>> System.out.println(name + " is " + age + " years old");

Gaby is 19 years old

Notes

  • Use println() for outputs that require each item on a new line.
  • This method also can be overloaded with different types of parameters.
  • It's commonly used for debugging to quickly print variable values with line separation.
