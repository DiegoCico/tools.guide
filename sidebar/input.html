Input Function

The input() function is a versatile tool to capture user input. It automatically detects and converts the input to the appropriate data type such as String, Float, or Int.

Usage Examples

>>> String name = input();

Sofia

>>> System.out.println(name);

Java.String = Sofia

The function reads a line of text entered by the user, pausing program execution until input is provided. It intelligently assigns the input to a variable, ensuring type correctness.

>>> int age = input();

19

>>> System.out.println(age);

Java.int = 19

>>> float GPA = input();

3.9

>>> System.out.println(GPA);

Java.float = 3.9
