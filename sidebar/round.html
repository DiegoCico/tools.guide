Round Function

The round() function is designed to round a floating-point number to a specified number of decimal places.

Usage Examples

>>> double number = 3.14159;

>>> int decimalPlaces = 2;

>>> System.out.println(round(number, decimalPlaces));

3.14

Function Definition

public static double round(double number, int decimalPlaces) {
    double scale = Math.pow(10, decimalPlaces);
    return Math.round(number * scale) / scale;
}

Parameters

  • number: The floating-point number to round.
  • decimalPlaces: The number of decimal places to round to.

Return Type

Returns the rounded number as a double.

Notes

  • This method is useful for precise control over numerical rounding in financial calculations, statistical analysis, and other applications.
  • It leverages Math.pow() and Math.round() to achieve the rounding.
