linesToString Function

The linesToString() function is designed to process an array of strings. It effectively concatenates each element into a single string, separating them line by line for improved readability and presentation.

Functionality Overview

This function transforms an array of strings into a well-structured text block, where each array element is placed on a new line. This is particularly useful in scenarios requiring formatted output from an array. Below is a practical example:

>>> linesToString(["Hello", "Goodbye", "Adios!"]);

Hello

Goodbye

Adios!
