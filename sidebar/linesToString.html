linesToString Function

The linesToString() function takes an array of strings and concatenates them, placing each element on a new line.

Usage Examples

>>> String[] lines = {"Line 1", "Line 2", "Line 3"};

>>> System.out.println(linesToString(lines));

Line 1
Line 2
Line 3

This function is particularly useful for formatting multi-line text from an array of strings.

Function Definition

public static void linesToString(String[] list) {
        for (String str : list) {
            System.out.println(str);
        }
    }

Parameters

  • lines: An array of strings to be concatenated.

Return Type

Returns a single string with each array element on a new line.

Integration Example

Combining with File Writing:

Integration Example with File Writing

String[] lines = {"Header", "Data1", "Data2"};
String content = linesToString(lines);
Files.write(Paths.get("output.txt"), content.getBytes());

Output in output.txt:

Header
Data1
Data2

Notes

  • Utilizes String.join() with the system's line separator for compatibility across operating systems.
  • Can be integrated with file writing, logging, or any other scenario requiring formatted text output.
